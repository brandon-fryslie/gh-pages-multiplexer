## Tech Debt Aggregate

See frontmatter `tech_debt`. Highlights:

1. **Phase 4 missing dedicated VERIFICATION.md** — only structural artifact gap. Recommend backfill (low effort, 04-02-SUMMARY already contains all the evidence).
2. **Two bugs caught + fixed during Phase 5 live validation** (commit `af8c889`):
   - `prepareBranch` ran `git fetch origin gh-pages --depth=1` in source repo cwd, silently shallowing the source clone and breaking metadata-extractor on PR reruns. Fixed: drop `--depth=1`.
   - `isShallowRepo` treated empty `.git/shallow` as shallow, but `actions/checkout@v4 fetch-depth:0` can leave that file empty after unshallowing. Fixed: present-but-empty = not shallow.
   - Root cause is the implicit cwd contract between branch-manager and metadata-extractor. Worth a retro to consider promoting `sourceRepoDir` shallow-state to an explicit field on `DeploymentContext`.
3. **action.yml node24 migration TODO** — D-01 deferred decision, runs.using=node20 today.
4. **v2 backlog (out of milestone, tracked):** VMGT-01..05 (default-version redirect, delete, alias, retention, sort), NAVE-01..02 (same-page nav, outdated banner). Custom-domain CNAME PR-preview URL slug not validated live (no CNAME in test repo).
5. **PR-commenter D-19 swallow boundary** — single documented exception (`is403` warn-only). Revisit if v2 broadens permissions.
